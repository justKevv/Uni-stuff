position: static;

The static position is used to set the element to static by default. The element will follow the normal position by default, the element is not affected by properties like top, bottom, right, and left.:

This element is styled using posision: static;




position: relative;

An HTML element using position relative will be located at its normal position. Setting the top, right, bottom, and left properties of an element positioned with "relative" will move it away from its normal position, Other content will not adjust to fit the gap left by that element.

This element is styled using position: relative;



position: fixed;

An HTML element that is set with position fixed will have a fixed property, without any changes even if the website page is scrolled. The settings for left, bottom, top, and right apply to position fixed:

This element is styled using position: fixed;















posision: absolute;

Element HTML yang menggunakan position absolute akan diposisikan relatif dengan element lain yang mendahuluinya yang terdekat, bukan relatif terhadap layar secara normal.:

This element is styled using position: relative
This element is styled using position: absolute;
